Allows the user to add a new staff or change information for a selected member.

About

In order to develop this project, we first had to learn the design patterns, I have studied and applied one GRASP pattern, which stands for General, Responsibility, Assignment, Software, Patterns and one GoF pattern that stands for Gang of Four. In my case, I decided to use Controller Pattern more specifically Role Controller, that is very popular in OOP (Object-oriented programming). I have also applied the FactoryMethod for the Gang of Four patterns. This pattern provides one of the best methods when we need to create objects but giving the choice of which class to instantiate, to the subclasses. For testing, I have used JUnit framework.

Technology

Java, MySQL, NetBeans and JUnit.
